Project description:	A side scrolling Pokemon game developed in C++ and Cocos2dx. The goal is to ride through the forest and capture Pokemon. Articuno is a smart sprite that runs from the player. To move, press the arrow keys. To fire Pokeballs press the spacebar button.

To Generate apk
1. Download Android Studio http://developer.android.com/sdk/index.html
2. Import your project 
3. Click the dropdown menu in the toolbar at the top
4. Select "Edit Configurations"
5. Click the "+"
6. Select "Gradle"
7. Choose your module as Gradle project
8. In Tasks: enter “assemble”
9. Press Run
10. After, the apk will be located in projectName\app\build\outputs\apk
