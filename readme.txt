Project description:	A side scrolling Pokemon game developed in C++ and Cocos2dx. The goal is to ride through the forest and capture Pokemon. To move, press the arrow keys. To fire Pokeballs press the spacebar button.

Contributions: 

Evan - Shooting, sprite sheets, music, merged program code, Made edits to fix Parallax scrolling, AI, established Git Repo, wrote readme

Chris - Parallax scrolling: The game scrolls forever. The farther back something is the slower it moves. 
Painters Algorithm: The small trees behind the larger trees, as well as the clouds above the forest to give the 2D game a sense of depth. 
AI: SmartPokemon Sprites
